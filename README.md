스프링부트 + Thymeleaf 헬스 케어 커뮤니티 사이트

## 📝 프로젝트 소개
바쁜 현대사회 속 현대인의 건강한 라이프스타일 촉진을 위한 정보 및 소통 서비스를 제공하는 사이트입니다.
